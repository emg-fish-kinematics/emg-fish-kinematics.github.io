# EMG Fish Telemetry Project Website

Static GitHub Pages website for:

**EMG-Driven Telemetry and Inference System for Fish: Pose Reconstruction and Flow Sensing**

## Deploy on GitHub Pages

1. Create a repository named `<your-username>.github.io` or any repository with Pages enabled.
2. Upload all files in this folder.
3. In GitHub, go to **Settings → Pages**.
4. Set the source to the `main` branch and root folder.
5. Visit the Pages URL after GitHub builds the site.

## Files

- `index.html` — one-page project website
- `style.css` — responsive styling
- `script.js` — tabs, BibTeX copy button, and simple results chart
- `assets/` — paper PDF and extracted figure images

Replace placeholder code/data links in `index.html` when your repository or dataset is public.
